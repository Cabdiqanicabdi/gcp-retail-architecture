# 🛍️ ShopMax GCP Retail Architecture
> Enterprise retail cloud architecture built on Google Cloud Platform

## 🎯 The Business Problem
ShopMax gets 10,000 customers daily but on Black Friday traffic jumps to 500,000. The website crashes, orders get lost, customers leave angry.

## ✅ The Solution I Built
A fully scalable, secure, event-driven architecture on GCP that auto-scales and never loses an order.
